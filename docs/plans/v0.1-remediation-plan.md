## Approach (Phase 0)

few shared files (`cmd/setup.sh`, `lib/store.sh`, `cmd/doctor.sh`, `bin/agent-secrets`, `install.sh`,
`README.md`, `SECURITY.md`, `lib/help.sh`) — parallel teammates would serialize on single-owner-per-file
anyway and risk merge conflicts on the docs↔code coherence. Gate (`shellcheck` + `bats` + telemetry) after
each commit. `bin/agent-secrets` is Read-tool-denied (basename) → read via `git show HEAD:bin/agent-secrets`,
edit via a `/tmp` script + Bash.
